Kommunikation

Wird bei einer Messstelle eine Kilowattstunde entnommen, dann entsteht eine Schuld von 23,718 Cent.

Umkehrfunktion

Werden 23,718 Cent bezahlt, wird als Gegenleistung eine Kilowattstunde an die Messtelle geliefert
