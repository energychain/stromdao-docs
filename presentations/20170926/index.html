Die Blockchain ist als Technologie geeignet, wenn über möglichst viele Akteure eine allgemeine Einigkeit über den Besitz von Digital Assets bestehen muss.

Die Blockchain ist als Technologie ungeignet, wenn kein Rahmen gebildet werden kann.

Konsens - Grid Actor

  • Abgaben, Steuern, Umlagen
  • Zeitsynchrone Ein/Ausspeisung
  • Einhaltung der Netzkonformität

  • ...

Jedes Kriterium, um ein autonom aggierender Grid Actor zu sein, kann auch als Zertifikat ausgedrückt werden. Bei der Blockchain Technologie wird dies mit sogenannten Tokens (Coins) abgebildet.

Tokens

Beispiel: Grünstrom

Damit B an A Grünstrom
liefern kann,
muss B im Besitz von
Grünstromtokens sein,
die von C herausgegeben werden.

Im Konsensrahmen ist gegeben, dass sowohl A als auch B auf C vertrauen.

Tokens

  • sind SmartContracts
  • Haben eine Regel, wie sie emitiert werden
  • Haben eine Regel, wie sie weitergegeben werden können
  • Erlauben die Nachweisführung
  • sind native Digtal Assets
  • können keinen negativen Besitz anzeigen.
